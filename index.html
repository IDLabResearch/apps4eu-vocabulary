Apps for Europe ontology

This is a vocabulary to describe Apps for X events. It consists of a vocabulary that describes the different app concepts, in the same way that DCAT describes datasets. This vocabulary includes properties that describe the datasets used, the applications' formats and access methods, the criteria used for judging the applications, the app contest it was part of, etc. The objective is that all the Apps for X contests in Europe use the same vocabulary in order to describe those concepts.

Ontologies reused

  • Dublin Core
    The Dublin Core Metadata Initiative (DCMI) Metadata Terms vocabulary defines general metadata attributes such as title, creator, date and subject.

  • FOAF
    The Friend Friend (FOAF) vocabulary defines terms for describing people, their activities and their relations to other people and objects

  • Creative Commons
    The Creative Commons (CC) schema defines terms for describing copyright licences.

  • Schema.org
    The Schema.org defines a collection of schemas.

  • Event Ontology
    The Event ontogy deals with the notion of reified events.

  • LODE (Linking Open Descriptions of Events)
    The LODE ontology describes an ontology for publiushing descriptions of historical events as Linked Data and for mapping between other event-related vocabularies and ontologies.

  • DCAT
    The DCAT ontology describes a schema to describe datasets in data catalogs.
